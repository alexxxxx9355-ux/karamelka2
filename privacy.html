Политика конфиденциальности сервиса «Карамелька AI»

Действует для сайта: https://karamelkafood.ru и Telegram-бота: t.me/karamelkafood_bot

Дата последнего обновления: 04.12.2025

1. Общие положения

Настоящая Политика описывает порядок обработки данных пользователями сервиса «Карамелька AI» (далее — «Сервис»).

Под «Администрацией» понимается лицо, владеющее и управляющее Сервисом.

Сервис работает на базе Telegram-бота и не осуществляет сбор избыточных персональных данных.

Используя Сервис, пользователь даёт согласие на обработку данных в соответствии с настоящей Политикой и законодательством РФ (152-ФЗ).

2. Какие данные собираются

Сервис собирает только технические данные, автоматически предоставляемые Telegram API:

  • Telegram ID;
  • username, имя в профиле;
  • язык интерфейса;
  • данные о действиях внутри бота (команды, клики по кнопкам, статистика использования);
  • использование функций (планирование питания, рефералка, премиум, участие в розыгрышах);
  • информация о платежах (факт оплаты, тип подписки — без банковских реквизитов).

Сервис не запрашивает и не хранит:

  • ФИО;
  • телефон;
  • email;
  • паспортные данные;
  • адрес;
  • банковские реквизиты.

3. Цели обработки данных

Данные используются только для работы Сервиса:

  • создание и поддержка аккаунта;
  • предоставление функций (ИИ-помощник, планирование, рецепты);
  • корректная работа премиум-подписки;
  • участие в реферальной системе;
  • участие в розыгрышах;
  • аналитика активности (в обезличенном виде);
  • предотвращение злоупотреблений (спам, многоаккаунтность).

4. Использование OpenAI/OpenRouter

Часть запросов пользователей может отправляться в OpenAI/OpenRouter для обработки ИИ-моделями.

Передаются только текстовые запросы пользователя, без данных аккаунта.

Администрация не несёт ответственности за обработку информации третьими сервисами.

5. Передача данных третьим лицам

Данные не передаются третьим лицам, за исключением:

  • OpenAI/OpenRouter — обработка текстовых запросов;
  • ЮKassa — обработка платежей (Сервис не имеет доступа к картам).

6. Хранение данных

Данные хранятся в базе данных Telegram-бота в течение всего срока использования Сервиса.

Данные удаляются:

  • по запросу пользователя;
  • при удалении аккаунта;
  • при прекращении работы Сервиса;
  • автоматически через 12 месяцев неактивности пользователя.

7. Удаление данных

Пользователь может запросить удаление всех своих данных через команду поддержки в боте или через контакты, указанные в разделе 10.

Удаление производится в течение 30 дней с момента обращения.

8. Использование cookies

Сайт использует только технические cookies для корректной работы (шрифты, стили, CDN).

Telegram-бот cookies не использует.

Мы не используем аналитические cookies и трекеры рекламных сетей.

9. Обновления политики

Администрация может обновлять Политику. Новая версия вступает в силу с момента её публикации на сайте или в интерфейсе Сервиса.

Продолжение использования Сервиса после обновления означает согласие с новой редакцией Политики.
